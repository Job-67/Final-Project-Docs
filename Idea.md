### core (หลัก)

- [ ] Game-Based Learning (การเรียนรู้ที่ใช้เกม)
    - Rules / Goal / Score / EXP / Level (กฎ / เป้าหมาย / คะแนน / ประสบการณ์ / ระดับ)
    - Fail → Feedback → Retry (ล้มเหลว → ข้อมูลป้อนกลับ → พยายามใหม่)

- [ ] Gamified Learning (การเรียนรู้แบบเกมิฟิเคชัน)
    - Badge / Achievement (ป้าย / ผลสำเร็จ)
    - Progress / Quest / Mission (ความก้าวหน้า / ภารกิจ / ภารกิจ)

- [ ] Interactive Learning Simulation (การจำลองการเรียนรู้เชิงโต้ตอบ)
    - Scenario-based (ใช้สถานการณ์)
    - Branching Path (เส้นทางแตกแขนง)

- [ ] Chat Interactive System (ระบบการสนทนาเชิงโต้ตอบ)
    - Mentor Bot / Debug Bot (บอทที่เป็นพี่เลี้ยง / บอทแก้ไขข้อผิดพลาด)

- [ ] Adaptive Difficulty (ปรับระดับความยาก)
    - ปรับระดับตามความเข้าใจผู้เรียน

### content (เนื้อหา)

- [ ] Thinking & Problem Solving (การคิดและแก้ปัญหา)
	- Think → Try → Fix (คิด → ลงมือทำ → แก้ไขปัญหา)
	- Predict Output (ทำนายผลลัพธ์)
	- Decompose Problem (แยกปัญหา)
	- Find the Bug (หาข้อผิดพลาด)
	- Explain Logic (อธิบายตรรกะ)

- [ ] Interactive Activity (กิจกรรมเชิงโต้ตอบ)
	- Drag-Drop Flowchart Symbol (ลาก–วาง สัญลักษณ์ผังงาน)
	    (Start / Process / Decision / Loop / End) (เริ่มต้น / ประมวลผล / ตัดสินใจ / วนซ้ำ / จบสิ้น)
	- Arrange Algorithm (เรียงลำดับอัลกอริทึม)
	- Fill Missing Command (เติมคำสั่งที่หายไป)
	- Match Concept ↔ Code (จับคู่แนวคิด ↔ โค้ด)
	- True / False Logic Check (ตรวจสอบตรรกะจริง/เท็จ)

- [ ] Game Mechanics (กลไกเกม)
	- Mini Puzzle (ปริศนาเล็ก)
	- Time Pressure Challenge (ท้าทายเวลา)
	- Limited Attempt (พยายามจำกัด)
	- Hint Exchange (แลกคำใบ้ด้วย EXP)
	- Boss Level (ระดับบอส - รวมหลายแนวคิด)

- [ ] Communication & Collaboration (การสื่อสารและความร่วมมือ)
	- Chat-Based Learning (การเรียนรู้ผ่านการสนทนา)
	- Peer Discussion Mode (โหมดการอภิปรายกับเพื่อน)
	- Pair Debugging (แก้ไขข้อผิดพลาดเป็นคู่)
	- Compare Solution (เปรียบเทียบวิธีแก้ไข)
	- Reflection Question (คำถามเพื่อการไตร่ตรอง)

- [ ] Assessment & Feedback (การประเมินและข้อมูลป้อนกลับ)
	- Instant Feedback (ข้อมูลป้อนกลับทันที)
	- Concept Check (ตรวจสอบแนวคิด)
	- Logic Check (ตรวจสอบตรรกะ)
	- Reflection (การไตร่ตรอง)
	- Progress Report (รายงานความก้าวหน้า)
